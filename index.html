Project Utama

Project ini menjadi bagian utama yang berkaitan langsung dengan mata kuliah Semantik Web.

CineMatch Semantic

CineMatch Semantic adalah aplikasi Web Semantik yang digunakan untuk menampilkan dan mencari data film. Data film direpresentasikan menggunakan RDF, struktur pengetahuannya dirancang menggunakan OWL, dan proses pengambilan data dilakukan menggunakan SPARQL.

Project ini juga menerapkan konsep Linked Data untuk menghubungkan data lokal dengan sumber eksternal. Dalam website profil ini, CineMatch Semantic direpresentasikan sebagai entitas Project yang memiliki relasi creator dan about.
