CineMatch Semantic adalah project aplikasi Web Semantik yang digunakan untuk menampilkan dan mencari data film. Data film direpresentasikan menggunakan RDF, struktur pengetahuannya dibuat menggunakan OWL, dan pencarian data dilakukan menggunakan SPARQL.
